Cross-platform Windows/Linux

Goal Rush is a fast paced arcade game where two-robots team will face each other to score the maximum of goals during short matchs.


HOW TO RUN THE GAME
---------------------------------------------------------------------

run goalrush.exe

If the game does not start you may need to install openal. Run openal/oalinst.exe on Windows or install libopenal on linux.

For a better gaming experience using dual stick controllers is recommanded.
